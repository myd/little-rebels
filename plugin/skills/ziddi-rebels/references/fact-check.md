# Fact-Check — Full Instructions (Step 5)

Read this file in full before running a fact-check. Run through every verifiable claim before anything else. Children's books get scrutinized closely — a wrong "first" or a wrong date erodes the book's credibility.

## Step 1 — Extract all claims

List every factual statement in the draft: dates, names, firsts, numbers, quotes, locations, events. Show this list to the author before evaluating anything.

## Step 2 — Check each claim

- State the claim as written
- State what you found: confirmed / contradicted / unverifiable
- Cite a specific source (URL, book title, archive) — not "multiple sources"
- If unverifiable: flag it explicitly. The author decides whether to cut it, soften it ("around 1953"), or find a primary source.

## Step 3 — Return a verdict

- ✅ All claims verified — ready to lock
- ⚠️ [N] claims need attention — listed below
- ❌ [N] claims are incorrect — corrections below

Never clear a draft if any claim is unverified. The standing reference case: a "widely accepted" claim that Bhanumati Ramakrishna was the first Indian actress to play a dual role turned out to be wrong — Nargis Dutt did it first in 1952. If it's not sourced, it doesn't go in.

## Without web access

If you cannot browse the web, a fact-check can still be useful — but it changes character. Follow the "If You Cannot Browse the Web" rules in SKILL.md, plus:

- State up front: *"I can't check live sources from here, so this is a consistency check against my training knowledge, not a true fact-check."*
- You may mark a claim ❌ only when your knowledge clearly contradicts it; everything you cannot positively confirm gets ⚠️, never ✅
- Never return the "✅ All claims verified — ready to lock" verdict without live sources — the best available verdict is ⚠️
- Recommend the author verify the flagged claims using the Go Deeper sources from the research brief

## Proactive trigger

Whenever the author shares a new or revised draft, ask: *"Want me to run a fact-check on this before we move to layout?"* Don't wait to be asked.
